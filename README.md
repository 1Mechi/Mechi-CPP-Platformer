C++ Game Development Sample – Raylib Platformer.

Name: Wisdom Amechi Onyeokeh

This project demonstrates my C++ programming skills using raylib.
It was built from scratch as part of my preparation for MSc Game Computing.

Features:
- Player movement with gravity and jumping
- Keyboard and controller input support
- Tilemap collision using image-based collision maps
- One-way platform collision system
- Basic animation system (idle, run, jump states)
- Basic health and death system
- Simple UI feedback (health display, game state)
- Collectible system (item pickup logic)


Focus Areas:
- Object-Oriented Design (Player, BaseChar, Obstacles)
- Real-time game loop implementation
- Physics handling (gravity, velocity, grounded state)
- Collision detection and resolution (tilemap + platforms)
- Incremental system building (movement → animation → collision → interaction)
